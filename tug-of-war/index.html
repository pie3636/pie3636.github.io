How does it work?
Regularly (usually when a get is reached), I fetch all the recent count data from Reddit using a small script that I wrote. After making sure everything is right, I then use another script to generate the statistics, that I simply paste into a file. This website then reads this data and displays the statistics in a more aesthetically pleasing way.
What's next?
I plan on adding a lot more statistics over the next few weeks. In particular, more user-based stats (who is usually the fastest, etc). Now that I already have the complete dataset, it is easy to generate those statistics. I also plan on making the site more easy to maintain, although that shouldn't affect the existing design. I might also make the graphs dynamic, though that takes some work.
Is the data reliable?
All the counts were gathered automatically, and then checked, so this isn't a source of error. However, some human error could be found in the data. For example, it sometimes happened that someone missed a count or counted twice the same number, and it wasn't always noticed. I could unfortunately not leave that as is, due to how the scripts I'm using work, and I had to manually "fix" a few counts here and there, by changing a digit here and there, or sometimes adding or removing a count. This is why if you examine the data carefully, you might sometimes find some discrepancies. I did my best to keep that edition to a minimum and do it in the most coherent way possible, but if you notice that something looks wrong, feel free to message me and I will see if I can change it.
No graph captions?
I am quite aware that in general, all graphs should be captioned. The reason why those aren't is simply because I'm using a spreadsheet to generate them, and I then manually take screenshots that I upload to imgur, before embedding it into the site. Since those screenshots can only have the same resolution as my screen, and there is a lot of information to convey in each graph, I try to make the actual graph as big as possible, to have the maximum resolution. This means eliminating everything that's on the sides. If you hover over a graph, you will see a tooltip explaining more in detail what each graph contains. In the future, I will most likely add web-based captions, or switch to dynamic charts.
