Designing webpages takes a lot of time that I'm not willing to take right now. So while I will probably make this page look better later, here is just a basic list of the subpages you can access on my GitHub page as of now:
  • Color Madness protocol. This is a server-client communication test for a project I've been working on and off on for a long time. This page won't work unless the server is running, which it probably isn't at the time you're reading this.
  • Color Madness canvas. This is a graphics test for the same game. This page will only partially work since the server and database are not currently up.
  • Count elegance calculator. This is a simple script that calculates how "pretty" a solution for a reddit count is (check reddit.com/r/counting, threads "Four fours" and "Counting with 12345" to understand what this is about).
  • Count solver. This is a work-in-progress script that solves a reddit count (see above for an explanation). This is a complex project that's been moved to an offline version for now.
  • Tower of Hero simulator. This page simulates many runs on the incremental mobile game Tower of Hero (check it out, it's awesome) in order to determine how different pieces of equipment are likely to loot, and in what amount. I regularly update this page to bring it up to date to the latest game version. The gold chest feature is currently not working.
  • Tug of War statistics. This page contains tons of statistics for the Reddit "Tug of War" thread in reddit.com/r/counting. I regularly update it with the latest data and add new statistics and graphs.
  • Fire Watcher. This is a work-in-progress incremental game I've been developing for some time. It has very little content as of now and a few minor bugs.
  • Kadok. A presentation webpage made for an English class project.
